##############################################################################
### Bash常用快捷键
##############################################################################

CTRL+R              # 历史命令反向搜索，使用 CTRL+G 退出搜索
CTRL+S              # 历史命令正向搜索，使用 CTRL+G 退出搜索
CTRL+U              # 删除字符到行首
CTRL+W              # 删除光标左边的一个单词
CTRL+X CTRL+X       # 连续按两次 CTRL+X，光标在当前位置和行首来回跳转
